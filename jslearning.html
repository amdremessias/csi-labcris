Por que essa aula importa

Se as variáveis são os blocos de construção, as funções, condicionais e loops são a lógica que faz o seu site se mover. Sem eles, o código apenas guarda dados parados. Com eles, você toma decisões automatizadas e processa listas de freelancers ou projetos num piscar de olhos.

Funções: Blocos de Lógica Reutilizáveis

Evite repetir código criando funções funcionais. Em JavaScript moderno, utilizamos a sintaxe tradicional e as famosas Arrow Functions (funções de seta).

// Função Tradicional
function calcularOrcamento(horas, valorHora) {
    return horas * valorHora;
}

// Arrow Function (Moderna e enxuta)
const calcularTaxaPlataforma = (valorTotal) => valorTotal * 0.05;

console.log(calcularOrcamento(10, 50)); // 500
console.log(calcularTaxaPlataforma(500)); // 25

Condicionais: Tomando Decisões (if, else e switch)

As condicionais guiam o fluxo da sua aplicação baseado em estados lógicos verdadeiros (truthy) ou falsos (falsy).

const saldoFreelancer = 150;

if (saldoFreelancer >= 200) {
    console.log("Saque liberado!");
} else if (saldoFreelancer > 0) {
    console.log("Saldo abaixo do mínimo de saque.");
} else {
    console.log("Sem saldo disponível.");
}

Loops: Automatizando Repetições (for, while)

Loops servem para varrer listas ou repetir tarefas iterativas sem precisar duplicar linhas de comandos manuais.

const tecnologias = ["React", "Node.js", "Tailwind"];

// Loop clássico for...of (Excelente para percorrer arrays)
for (const tech of tecnologias) {
    console.log(`Habilidade requisitada: ${tech}`);
}

Cheatsheet

Estrutura Uso Comum
() => {} Arrow Function moderna para funções rápidas.
if / else Validar regras de negócio ou permissões.
for...of Percorrer elementos de uma lista de forma simples.
